Esta é uma página HTML com o objetivo de praticar o uso das seguintes *tags*:
~~~
- <html></html>
- <h1></h1> até <h6></h6>
- <p></p>
- <article></article>
- <span></span>
~~~
Além da organização do *body* HTML em:
~~~
- <header></header>
- <nav></nav>
- <section></section>
- <aside></aside>
- <footer></footer>
~~~
